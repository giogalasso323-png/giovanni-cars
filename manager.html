Connect to Google Sheets

One-time setup — your inventory will live safely in your Google Drive, accessible from any computer.

1

Create a new Google Sheet

Go to sheets.new in your browser. Name it Dublin Toyota Inventory.

2

Open Apps Script

In the sheet, click Extensions → Apps Script. Delete everything in the editor. Paste in the code from apps-script.js (in your download). Click Save (floppy disk icon).

3

Deploy as a Web App

Click Deploy → New deployment. Click the gear icon and choose Web app. Set Execute as → Me. Set Who has access → Only myself. Click Deploy, authorize when prompted, then copy the URL it gives you.

4

Paste your Web App URL here

Looks like: https://script.google.com/macros/s/…/exec
